# Método Honorário Seguro — Página de Vendas

Este repositório contém a página de vendas oficial do **Método Honorário Seguro**, um sistema completo desenvolvido para contadores que desejam acabar com a inadimplência em seus escritórios e garantir o recebimento de honorários no prazo.

## 🚀 Sobre o Projeto

A página foi construída com foco em conversão, utilizando uma estrutura de copywriting persuasiva e design moderno, otimizada para visualização em dispositivos móveis e desktop.

### Principais Seções:
- **Headline Impactante:** Focada na dor do contador (inadimplência).
- **Lista de Dores:** Identificação imediata com os problemas do dia a dia.
- **Módulos do Método:** Detalhamento do que o aluno irá aprender.
- **Botões de CTA:** Integrados diretamente ao checkout da Hotmart.
- **Depoimentos e Prova Social:** Reforço de autoridade.

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3 (Custom Variables & Flexbox)**
- **Google Fonts (Inter)**

## 🔗 Link de Acesso

A página está hospedada via GitHub Pages e pode ser acessada pelo link público configurado na Hotmart.
